# Building a Simple Ken Burns Slider with Vue.js

[![Patreon](https://img.shields.io/badge/patreon-donate-blue.svg)](https://www.patreon.com/maoberlehner)
[![Donate](https://img.shields.io/badge/Donate-PayPal-blue.svg)](https://paypal.me/maoberlehner)

This is an example project for the following article: [Building a Simple Ken Burns Slider with Vue.js](https://markus.oberlehner.net/blog/building-a-simple-ken-burns-slider-with-vue/)

## Build Setup

```bash
# Serve with hot reload.
npm run serve

# Build for production with minification.
npm run build
```
